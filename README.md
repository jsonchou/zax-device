# ZAX Device Util
## there have 6 modes of amd, cjs, es, iife, system, umd. 
## support SSR Miniprogram Browser slide

## install

``` base
npm i zax-device -S
```

## build with rollup

``` base
npm run build
```

## use

``` javascript 
const zaxDevice = require('zax-device')
or
import zaxDevice from 'zax-device'
```
